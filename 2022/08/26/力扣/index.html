1.打家劫舍问题（动态规划DP）

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

给定一个代表每个房屋存放金额的非负整数数组，计算你 不触动警报装置的情况下 ，一夜之内能够偷窃到的最高金额。

启发

  1. 动态规划其实就是解决最小重叠子问题的一个方法。
  2. 动态规划的关键是找准迭代表达式（也叫状态转移方程）和初始条件（也叫基准）。
  3. 迭代表达式可以从递归中获取思路。
  4. 初始条件往往是递归从后向前的结束条件。
  5. 递归优化的过程，就是动态规划的逆向过程。
  6. 动态规划的空间复杂度往往是可以考虑优化的。

3.零钱兑换问题

给你一个整数数组 coins ，表示不同面额的硬币；以及一个整数 amount ，表示总金额。
计算并返回可以凑成总金额所需的 最少的硬币个数 。如果没有任何一种硬币组合能组成总金额，返回 -1 。
你可以认为每种硬币的数量是无限的。

规律：凑齐的钱数=1个硬币面值+（凑齐的总钱数-当前硬币面值）
那么：凑齐前钱数的硬币数=1个硬币+凑齐[总钱数-硬币面值]所需的最小硬币数
由于硬币面值有多个，因此：凑齐钱数的硬币数=[1个硬币+凑齐[总钱数-当前硬币面值]所需的最小硬币数]的最小值

    • 动态规划的基本思想

    什么问题可以用动态规划解决？

    1. 存在最优子结构，子问题相互独立。
    2. 遇到找不到最优子结构时，可以适当变换问题思路再来寻找。
    3. 巧妙的抓住，最值 这个关键字。

    解决动态规划问题可尝试的基本思路

    1. 如果一下子无法找到动态规划的关键，可以一步一步的去尝试。
    2. 可以先用最简单的思路写出暴力递归的方法。
    3. 然后用标志变量，记忆化搜索。
    4. 最后，找到合适的动态规划思路。

    动态规划的基本步骤

    1. 找准状态，简单来说，就是 DP[i] 的含义，你赋予它什么含义，就是什么含义。
    2. 找到状态转移方程，也就是迭代表达式，通常可以用具体的数据去打开你的思维。
    3. 找到基准条件，也就是初始条件，自下而上的一般是前几个值，自上而下的一般是后几个值。
    4. 使用合适的循环条件，将这一思想用代码呈现出来。

    动态规划的dp遍历

    1. 需要保证在你遍历时，你所用到之前的条件已经被正确计算。
    2. 需要保证你遍历的末尾，就是你所需要的答案。

    给你一个整数数组 coins 表示不同面额的硬币，另给一个整数 amount 表示总金额。
    请你计算并返回可以凑成总金额的硬币组合数。如果任何硬币组合都无法凑出总金额，返回 0 。
    假设每一种面额的硬币有无限个。 
    题目数据保证结果符合 32 位带符号整数。

    分析：对于面额为coin的硬币,当coin≤i≤amount时,如果存在一种硬币组合的金额之和等于i-coin,则在该硬币组合中增加1个面额为coin的硬币,即可得到一种金额之和等于i的硬币组合.因此需要遍历coins，对于其中的每一种面额的硬币，更新数组dp中的每个大于或等于该面额的元素的值。
